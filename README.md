新手指南
--------
从<http://www.dcloud.io/hellomui>下载Hello MUI并安装，可直接在手机上体验MUI的控件UI及能力展示；
若想详细了解Hello MUI的源码及工作原理，可从<http://www.dcloud.io>下载Hbuilder，选择新建“移动APP”，并选择“Hello MUI”工程模板，创建工程；然后通过数据线将手机连接上电脑，点击运行，也可以在手机上体验MUI的各项能力。
